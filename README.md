# 飞书低代码开发日志

## 项目简介
这是一个用于记录每日飞书低代码开发进度的项目。通过这个仓库，我们可以追踪和分享在飞书低代码平台上的学习和开发经验。

## 使用方法
1. 克隆仓库到本地
```bash
git clone git@github.com:ZUOLI11111111/feishu-hi.git
cd feishu-hi
```

2. 更新日志
   - 直接编辑README文件，添加你的开发进度
   - 提交变更并推送到远程仓库

## 提交指南
每次提交应包含：
- 日期
- 完成的功能或学习的内容
- 遇到的问题和解决方案（如果有）
- 下一步计划
